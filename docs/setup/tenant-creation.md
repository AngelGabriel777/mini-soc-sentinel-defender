# Tenant Creation – Mini SOC SC-200

## 1. Overview
Creation of the dedicated Microsoft Entra ID tenant for the Mini SOC environment.

## 2. Tenant Information
- Organization name: MiniSOC-SC200  
- Initial domain: minisocsc200.onmicrosoft.com  
- Region: Colombia  

## 3. Steps Performed
1. Accessed https://portal.azure.com  
2. Navigated to *Microsoft Entra ID > Manage tenants*  
3. Created a new Azure Active Directory tenant  
4. Switched to the new tenant  
5. Verified and attached the Azure subscription to this tenant
